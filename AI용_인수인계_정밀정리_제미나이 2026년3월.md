# 24시열린의원 EMR(의사랑) 처방 마스터 데이터베이스 및 알고리즘

## [Part 1] EMR 시스템 조작 및 청구 기본 알고리즘
### 1. 코드 및 오더 배열 규칙
* **상병 배열:** ① 만성질환(cd 청구용) ➔ ② IM 제제 연결 상병 ➔ ③ 주상병 ➔ ④ 기타 필요 상병
* **처방 배열:** ① IM 제제 ➔ ② IV 제제 ➔ ③ PO 약제(만성 ➔ 일반) ➔ ④ 외용제(연고, 패치, 시럽 등) ➔ ⑤ 기타 검사

### 2. EMR 조작 단축키 및 주의사항
* **타 진료실 인계:** 대기 환자 우클릭 ➔ [다른 진료실로 보내기] ➔ 2진료실 선택
* **비호출 차트 확인:** 대기 환자 우클릭 ➔ [유비패널 비호출 조회]
* **외용제 일수 주의:** [Ctrl+T]로 투약일수 일괄 변경 시, 외용제(시럽 `~sy`, 패치 `pat1~3`, 가글, 연고 등)는 변경되지 않으므로 개별 수동 변경 필수.
* **가루약 조제료 산정:** 6세 이상 가루약 처방 시, 처방 코드 우클릭 ➔ [가루약] 반드시 체크.
* **배제된 상병(Rule Out):** 의심 검사 시 오더 입력 후 우클릭 ➔ [배제된 상병] 처리. (확진 시 해당 날짜로 돌아가 해제 후 메인 상병으로 변경).

### 3. 주사제(IM/IV) 삭감 회피 룰셋
* **기본 원칙:** 본원 주사제는 전원 'IM' 코드로 기본 등록됨. 
* **2종 이상 처방:** 1종류만 급여(보험) 인정. 나머지는 코드 뒤에 `-b`를 붙여 비급여 추가 주사제로 입력.
  * 예시: `tra`(보험) + `busb`(비급여) / `ns` + `macb` + `ranib` + `trb` + `tirop`
  * 비급여 묶음 처방 탭: `inj4 비급여추가주사제`

## [Part 2] 만성질환 (Chronic Diseases) 상세 데이터
* **재진 산정 기준:** 투약 종료일 이후 90일 이내면 재진. 90일 초과 시 초진. (월 2회, 연 12회 `cd` 만성질환관리료 청구).

### 1. 당뇨병 (E10~E14)
* **검사 코드:** BST 단독 확인은 `bst` 오더. 혈액검사는 `glu`.
* **명세서(MX999):** 초진 재처방 시 `<당뇨약 재처방>`. 증량/2제 병용 시 `<HbA1c 7.0(또는 7.5) 이상으로 증량/추가>`.
* **주의:** 검사는 3~4개월 주기 (3개월 이내 삭감). 단순 검사 시 `r81`(당뇨), `r730-4`(당뇨병전기) 등 증상 코드를 잡고 '배제된 상병' 처리.

### 2. 고혈압 (I10~I15)
* **명세서(MX999):** 초진 시 `<BP 14X/9X로 혈압약 처방>`. 증량 시 `<수축기 140 or 이완기 90 이상 지속되어 증량>`. 타 성분 교체 시 `<기존 약 부작용>` 등 사유 명시.
* **병용 금기(삭감):** Diuretic+α Blocker / β Blocker+ACEi / β Blocker+ARB / ACEi+ARB
* **항혈전제 룰셋:**
  * **항혈소판제 2종 병용:** 무조건 삭감. (1종 F12 비급여 처리 또는 기존 병원 안내).
  * **항혈소판제 단독:** `i252`(오래된 심근경색), `i638`(기타 뇌경색) 필요. Aspirin 못 쓸 경우 특정내역에 `<Aspirin 알러지/부작용/저항성>` 명시.
  * **NOAC (자렐토, 엘리퀴스 등):** 보험 심사 까다로움. F12 비급여 처리 또는 전원.

### 3. 이상지질혈증 (E78)
* **특징:** 만성질환관리료(`cd`) 비대상 질환.
* **명세서(MX999):** 초진 및 약 증량/추가 시 1년 이내 검사 결과(LDL 등) 및 위험요인(가족력, 흡연 등) 필수 기재. (본원 유지 처방 시 기재 생략).

### 4. 골다공증 (M818)
* **명세서(MX999):** 매 방문 시 1년 이내 BMD 결과(T-score) 기재 필수.
* **투여 주기:** Alendronate(1주 1회), Risedronate(1달 1회), Ibandronate(3개월 1회).
* **특이사항:** `vitd`(시트라칼에프), `디카맥스`는 골다공증 상병(`m818`) 단독으로 급여 인정.

### 5. 편두통 (G439) / 불면증 (G470, F510)
* **편두통:** `cd` 청구 가능. 
  * 약속처방: 싸리움(예방), 이미그란(급성), 크래밍(급성) 중 1종만 사용.
  * 크래밍(`Craming`): 해열진통소염제 분류. 타 NSAIDs/타이레놀과 병용 시 **총 2종 이하**로 제한(3종부터 삭감).
  * 에피손(`epe`) 병용 시: `m542`(경추통) 상병 필수.
* **불면증:** `cd` 청구 가능. 
  * 스틸녹스(`zol`): 최대 28일 처방. 식욕억제제(디에타민 등) 등 타 향정 중복 절대 불가.
  * 상병 변경: 투약 종료 후 30일 이내(`g470`+`cd`) ➔ 30~90일(초진 변경 + `f510`) ➔ 90일 이후(초진 + `g470`).

## [Part 3] 호흡기 질환 (Respiratory) 상세 DB
### 1. 진해거담제 삭감 룰셋 및 병용 금기
* **보험 한도:** 성인(상기도 2종, 하기도 3종), 6세 미만(복합시럽 1종 추가 인정). 슈다페드(`suda`)는 카운트 제외.
* **병용 금기(삭감):** 동종 약리작용 중복 시 삭감.
  * **[중추성 기침]** `co`, `cosy` 
  * **[말초성 기침]** `drop`, `dropsy` 
  * **[거담제]** `ambsy`, `ac`, `erdo` 
  * ⚠️ **`drop`(엘피진) + `ac`(뮤시딘) 병용 삭감.** (반드시 1종 F12 비급여 처리).
* **특정 상병 요구 약제:**
  * `ac`, `ac2`: 후두염(`j040`), 부비동염(`j0180`), 기관지염(`j209`)
  * `umk`(펠라고니움): 기관지염(`j209`) - 체중 무관 정해진 용량 사용.
  * `atock`, `pat1~3`(패치): 기관지염(`j209`), 천식(`j459`)

### 2. 소아 체중별/연령별 세부 용량 (류코트리엔 조절제)
* **Lukasy (프란루카스트 건조시럽):** 
  * ~9kg: `4` / 10~14kg: `5` / 15~19kg: `6` / 20~24kg: `8` / 25~30kg: `10`
* **Luka (몬테루카스트 츄정/정):** 
  * 2~5세: 4mg 츄정 / 6~14세: 5mg 츄정 / 15세 이상: 10mg 정제

### 3. 천식 (Asthma, J459) 및 흡입제 디테일
* **류코트리엔 조절제 청구:** 특정내역(`JX999`) ➔ `<타 천식약제로 증상조절이 되지 않는 2단계 이상의 천식>` 기재.
* **흡입제 룰셋 (루틴 처방 시 PFT 결과지 요구됨):**
  * SABA: 벤토린 (에보할러, 네뷸, 레스퓰)
  * ICS: 풀미코트 (터부헬러, 레스퓰)
  * ICS+LABA: 심비코트 (라피헬러, 터부헬러), 세레타이드 (디스커스, 에보할러), 포스터 (HFA)
  * 분무용 용액(네뷸라이저)은 일반 흡입제 사용 곤란 시에만 인정(100/100 여부 확인).

### 4. 독감 (Influenza)
* **타미플루 캡슐:** 독감 폴더에서 직접 클릭 (과거 30/45/75 용량 코드는 미사용).
* **타미플루 현탁액(`tamisy`):** 체중당 산정. (예: 16kg ➔ 7.5ml x 2회 x 5일 = 처방창에 용량 15, 일투수 2, 일수 5 입력).
* **페라미플루(`tamiiv`):** 만 10세/중학생 이상 1회 2앰플 IV 투여.

## [Part 4] 소화기 질환 (Gastrointestinal) 상세 DB
### 1. 장염 체중별 처방 묶음 코드 (AGE)
* `a6` (6~8kg), `a9` (9~11kg), `a12` (12~14kg), `a15` (15~17kg), `a18` (18~20kg), `a21` (21~23kg), `a24` (24~26kg), `a27` (27~29kg)

### 2. 소화기 약물 상병 매트릭스 (O=급여인정, X=삭감)
| 약제 (성분) | 위염 | 장염 | IBS(k58) | 소화불량 | 구역/구토 |
| :--- | :---: | :---: | :---: | :---: | :---: |
| **tirop, bus, tr** (진경제) | O | O | O | - | - |
| **trime** (운동조절제) | O | O | O | O | O |
| **levo, mosa** (운동조절제)| X | X | O | O | O |
| **mac, macpo, dom** | - | - | - | - | O |
* **주의:** `macpo`/`dom`은 5일 이상 처방 시 삭감. `tirop`은 14세 이하 안전성 미확립.

### 3. 소화성궤양용제 및 PPI 룰셋
* **중복 삭감:** 방어인자(`reba`) + 공격인자(`cime`, `famo`, `ppi`) 병용 시 삭감. 라니티딘 처방 금지.
* **PPI 세부 규정:**
  * `ppi` (저용량): 기간 제한 없음.
  * `ppi2` (고용량): 첫 처방일 기준 **1년 내 8주까지만** 급여. (특이증상 F6에 누적 일수 기재).
  * `pcab` (케이캡 등): 1회 최대 4주 처방 급여. 1년 8주 이상 가능.
* **알지네이트(`al`):** 식도염(`k210-1`) 상병에만 급여. (단순 위염은 100/100). 식후 2~3시간 뒤 복용.
* **스멕타이트(`sme`):** 설사 동반 상병 필수. (IBS 처방 시 `k589` 불가, 반드시 `k581` 입력). 2세 미만, 임부, 수유부 금기. 대체약 `hidra`(하이드라섹산).

### 4. 수액 코드 (IV Fluids)
* `gw6` (리케어솔 250ml), `gw8` (가네파솔 8%), `gw10` (브레노솔 250ml), `gw15` (콤비플렉스엠시티페리주), `gw3` (멀티블루 믹스용, 단독 사용 지양).

## [Part 5] 감염 / 안과 / 특수 증상 세부 처방
* **방광염(`n300`):** 1차 `aug2` ➔ 2차 `cipro` (JX999: 1차 항생제 호전 없음) ➔ 3차 모누롤산.
* **소아 요로감염(UTI):** 1차 `cefa`/`augsy` ➔ 2차 바난시럽(3세대) ➔ 전원.
* **귀두포피염:** `cefasy` + `lacto2` + `typow` (3일) + `bac`(뮤피로신 연고).
* **수두(`b01`):** Acyclovir 경구 + 칼라민 로션. (법정 2군, 보건소 신고 명부 작성).
* **안과 안연고:** 안과 질환 외 상처 감염 예방 목적 사용 시 전액 본인부담(100/100) 적용. (타리비드, 베아오플, 네오덱스 등).
* **안구건조증:** `eyemini`(0.18% 미니) ➔ 용량 5, 일투수 1, 일수 12 세팅하여 최대 2통 권장.
* **빈혈(`d50`):** 1차 `훼로바유` ➔ 2차 `볼그레액` (JX999: 경구 위장장애 사유 기재).

### [특수 마이크로 증상 및 약제]
* **딸꾹질:** 위염 폴더 내 클로르프로마진, Baclofen 처방.
* **눈떨림:** 마그네슘(비급여) + 소화제(조제료 산정 목적 보험 추가).
* **식욕부진:** 메게이스현탁액 (1, 1, 5 비급여) 또는 Trestan (식욕부진 상병 + `mosa` 추가하여 조제료 보험).
* **부종:** 엔테론(림프부종), 다이크로짇(국소부종), 베니톨(정맥임파부전, 급성치질은 최대 7일).
* **전립선비대/방광:** 유리토스(`n319` 방광신경기능장애), 트루패스(`n400` 전립선증식).
* **메니에르병:** 메네스정(Betahistine) - 비만 식욕 조절용으로도 차용됨.

## [Part 6] 근골격계 (MSK) 및 외상(Dressing)
* **디클로페낙 주사(`d`):** `m79108`(기타 근통) 자동 입력됨 (삭제 금지).
* **근이완제(`epe`):** `m545`(아래허리통증) 또는 `m542`(경추통) 상병 필수. 안전성 위해 1일 2회(BID) 처방 권장.
* **쎄레브렉스:** 무릎관절증(`m170`) 등 명시된 관절염 코드 필수. 단순 급성 통증은 100/100.
* **트라마돌 병용 삭감:** `set`(울트라셋) + 타 NSAIDs/타이레놀 = 삭감. `semi`(울트라셋 세미) + 타 NSAIDs = 인정.
* **외상 드레싱(`dres`) 세부 재료:** 
  * `eb2` (탄력붕대 5cm), `eb4` (탄력붕대 10cm)
  * `form` (메디폼 1장), `coban` (코반 1인치, 수지부 0.2 입력)
  * `wd` (운드드레싱 6x7, 9x10 규격)

## [Part 7] 비급여 및 일반 약제 원칙
* **일반/비급여 처방:** 사후피임약(노레보원), 발기부전(비아그라), 비만(디에타민), 여드름(이소티논), 분실 재처방 등.
* **원칙:** 기존 보험 진료와 묶음 처방 절대 금지. **'일반' 재접수** 후 초진 처리하며 상병은 **`j00`(감기)**으로 일괄 통일. (F12 비청구 임의 처리 불가).
* **신규 약제:** 뮤라틱주(`aci`) - Ambroxol 대체 수액 믹스용 (비용 8,000원). 에피네프린(`epi`) - 응급 IM 폴더 구비.
