# 사후피임약

기본 차팅: 관계 이후 몇 시간 지났는지

마지막 복용 이후 정상 생리 주기 지났는지 여부

이전 복용 시 부작용 여부

주의점 설명
비고: - 생리 주기 마다 1회만 사용 가능
- 경구피임약 복용 중이었거나 이미 같은 주기에 사후피임약 복용한 경우에는 산부인과로 refer
-복용 후 3시간 이내 구토하면 다시 복용
-1~2일 가량 약간의 두통, 구역감, 부정출혈 있을 수 있음

-성공확률
24시간 이내 : 95%
48시간 이내 : 85%
72시간 이내 : 58%
상병: 일반 접수하고 초진 진찰료만 청구
(금액 얼마로 고정 없이 약값 + 그 날 초진진찰료로 청구)

상병은 j00감기 라든지 아무거나
