# AI용 인수인계 정밀 정리본 — 인수인계_2026년3월

## 0. 문서 성격 / 사용 전제
- 문서 목적: 24시열린의원 내부 인수인계용 실무 매뉴얼.
- 구성: 질환별 정리, 보험/비보험, 의사랑 사용 팁, 수액, 법령, 약품 정보 확인 포인트, 알러지/대체약, 진료비 청구/비청구.
- 검색 팁: 질환명/약명/계열/상병코드로 Ctrl+F 검색. 코드는 앞 두 자리 정도만 검색하면 더 정확함.
- 정리 방식: 질환명 → 관련코드 → 주요 내용 → 기타 관련내용 순.
- 처방 예시는 의사랑을 처음 다루는 GP 수준을 기준으로 작성.
- ‘만성’ 표시가 있는 항목은 만성질환 관련 공통 규칙을 함께 봐야 함.
- 미완성 인수인계 사이트(24시열린의원 notion.site)도 병행 참고.
- 병원 운영 특성상 스케줄 근무로 담당의가 자주 바뀌므로, 차팅과 오더의 통일성이 매우 중요함. 이전 기록 적극 참고 필요.

## 1. 전체 공통 원칙
### 1-1. loxoprofen 관련
- 2024년부터 loxo(loxoprofen)는 호흡기 상병에서 삭감.
- 그래서 ‘아래허리통증’ 코드를 연결상병으로 같이 잡는 방식 사용.
- 대체 NSAID로 펠루비프로펜, 부루펜, 덱시부프로펜 등 사용 가능.
- 추가 변경사항에도 동일하게 재강조: 감기 상병에서 loxoprofen 삭감 / 기본 NSAID는 dexi, bru / loxo는 요통 코드 연결 후 사용.

### 1-2. 기본 상병 순서
1. 만성질환-cd(만성질환관리료) 필요한 상병
2. IM 제제 관련 상병
3. 주상병
4. 기타 필요상병

### 1-3. 기본 오더 순서
1. IM 제제
2. IV
3. PO medication
4. 기타 검사
- PO는 만성질환 약 → 일반 약 순
- 외용제는 마지막

### 1-4. 주사제 입력 체계
- 본원은 IM용/IV용 주사제 입력이 따로 나뉘지 않고 모두 IM 제제로 등록.
- 따라서 2종 이상 주사제를 쓰면 추가 주사제는 비급여 등록 코드 사용.
- 묶음처방에서 해당 약제를 확인 가능.
- 디클로페낙을 제외하면 대부분 기본 등록 코드에 `-b` 붙인 코드가 비급여 추가 주사제.

### 1-5. 필수 사전 학습 항목
- 원외처방 코드와 심사기준
- 약 중복처방과 DUR
- R코드, 의증, 배제된 상병

### 1-6. 배제된 상병(rule out) 원칙
- 검사 당시 의심질환이지만 확정이 아닌 경우 ‘배제된 상병’ 처리 가능.
- 이렇게 하면 관련 검사 삭감을 피할 수 있음.
- 결과 확인 후 실제 질환이면 이전 검사 날짜 기록으로 들어가 해제.
- 이전 확진환자에는 쓰지 않음.
- 단순 확인 목적, 관련 과거력 없는 환자에서 사용.
- 메인상병으로는 사용 불가.

### 1-7. ranitidine 관련 변경
- 기존 `dige`(ranitidine + bismuth + sucralfate) 사용불가.
- `rani` ranitidine IM도 사용불가.
- 대체약:
  - Reba = rebamipide
  - Cime = cimetidine
  - famotidine 등도 가능하나 용법/용량 반드시 확인
- 속쓰림 심한 경우 ranitidine IM 대체 주사제로 ppiiv(판타졸) 고려.

## 2. 의사랑 / 근무 / 현장 운영
### 2-1. 진료실 운영 구조
- 1번 진료실이 메인, 2번 진료실이 보조.
- 2번 진료실은 공휴일, 평일 저녁, 독감철 등 환자 많을 때 운영.
- 모든 환자는 1번 진료실로 접수되고, 2번 진료실은 대기환자를 넘겨받아 처리.

### 2-2. 환자를 다른 진료실로 보내는 법
- 대기환자 우클릭 → ‘다른 진료실로 보내기’
- 보내려는 진료실 확인 → 확인
- 접수 진료실이 2진료실로 바뀐 뒤 해당 환자 진료

### 2-3. 호출 없이 차트 열기
- 가족/일행 이미 들어와 있음, 대기환자 미리 확인 필요 등 접수대 호출 없이 열어야 할 때:
- 대기 환자 우클릭 → ‘유비패널 비호출조회’
- 불필요한 호출/환자 호명 오류 방지

### 2-4. 접수대와 소통
- 의사랑 쪽지 기능도 있으나 실제로는 LIVE Talk를 주로 사용(의사랑 로그인 시 자동 실행)
- 환자 성명과 필요한 조치를 간략히 보내면 됨

### 2-5. 기타 의사랑 실무 주의
- Ctrl+T로 투약일수 일괄 변경 시 외용제/시럽/패치류(Cosy, hidra, pat1~3 등)는 같이 안 바뀔 수 있으므로 반드시 재확인.
- 6세 이상에서 약을 갈아 쓸 때 해당 약 우클릭 → ‘가루약’ 체크해야 조제 수가 인정.
- 약 보험 여부 판단, 100/100/비급여, 보험/일반접수, 각종 서류 발급, 의뢰서, 진단서/소견서, 진료확인서, 초진기록지, 건강진단서, 의료인용 서류, 차팅 방식은 별도 항목으로 반복 강조됨.
- 묶음처방 `3nonpay`는 비급여 수가 참고용.

## 3. 만성질환 공통 규칙
- 대상 질환:
  1) 고혈압(I10-I13, I15)
  2) 당뇨(E10-E14)
  3) 정신 및 행동장애(F00-F99, G40-41)
  4) 호흡기 결핵(A15-A16, A19)
  5) 심장질환(I05-I09, I20-I27, I30-I52)
  6) 대뇌혈관질환(I60-I69)
  7) 신경계질환(G00-G37, G43-G83)
  8) 악성신생물(C00-C97, D00-D09)
  9) 갑상선장애(E00-E07)
  10) 간질환(B18, B19, K70-K77)
  11) 만성신부전증(N18)
- 첫 방문 시 30일까지만 처방, 안정 시 90일까지 가능
- 의사랑 기본은 30일 내 방문만 재진으로 보지만, 만성질환은 마지막 투약 종료 후 90일까지 재진으로 수기 변경 필요
  - 예: 30일분 처방이면 마지막 내원 후 120일까지 재진 처리 가능
- cd(만성질환관리료) 청구 위해 관련 상병을 주상병(맨 위)으로 둬야 함
- cd는 1달 2회, 연 12회 가능
- 이상지질혈증은 cd 대상 아님
- 병원에서 주로 쓰는 만성질환성 상병 예:
  - f510 / g470: 수면제
  - F419-1: 불안 NOS → 인데놀, diazepam 등
  - G43: 편두통
  - E03 / E059: 갑상선 기능저하/항진
  - F067: 경도인지장애(글리아티린 처방 시, 60세 미만 금지/반드시 전화 확인)

## 4. 질환별 / 상황별 정리

### 4-1. 당뇨(만성)
- 코드: E10-E14, R81, R730-4
- 명세서:
  - 첫 방문만 `<당뇨약 재처방>` 기재
  - 증량 시 `<HbA1c OO 이상으로 당뇨약 증량>` 등 근거 기재
- 기준:
  - HbA1c 7.0 이상: 1단계
  - 7.5 이상: 2단계 또는 2제
- 검사:
  - HbA1c 3~4개월마다 (3개월 이내 재시행 시 삭감)
  - 신기능 1년마다
  - bst는 단순 당 체크, glu는 혈액검사
- 의심 환자 검사 시:
  - 당뇨 상병 입력 후 ‘배제된 상병’ 처리 가능
  - HbA1c는 당뇨 의증도 가능
- 확진 검사 무료 사업:
  - 1차의료기관 방문 시 bst + 진찰료 무료 사업 있음
  - 다른 검사 추가되면 보험접수 하나 더 필요, 하루 2회 청구라 지점장 보고
- 약 안 쓰고 확인만 하는 의심 환자:
  - r81 당뇨, r730-4 당뇨병전기 등 증상코드 사용해 추후 보험문제 방지

### 4-2. 고혈압(만성)
- 코드: I10, I11, I12, I13, I15, R030
- 명세서:
  - 타원 repeat 첫 방문은 `<혈압약 재처방>` 1회만 기재
  - 증량 시 `<BP 140/90 이상 지속되어 혈압약 증량함>`
  - 감량 시:
    - 같은 성분 감량은 재증량 전까지 별도 기재 불필요
    - 성분 교체는 부작용/효과부족 등 타당 사유 기재
  - 본원 첫 처방은 `<BP XXX/XX로 혈압약 처방함>`
- 병용 허용 조합 반드시 보험심사기준 확인
- 콩코르:
  - 5mg: 고혈압 상병 가능
  - 2.5mg: 심부전 상병 필요
- 단순 확인 환자:
  - R030 사용
  - ‘상세불명의 고혈압(의증)’은 보험 문제 소지
- 확진검사 무료 사업:
  - 진찰료 무료
  - 추가 검사 있으면 보험접수 추가 + 지점장 보고
- 초처방 30일, 안정 시 90일 가능

### 4-3. 항혈전제
#### 항혈소판제
- clopidogrel, cilostazol 등
- aspirin 없이 단독이면:
  - I252 오래된 심근경색증 또는 I638 기타 뇌경색증 등 관련 코드 필요
  - 보험 문제 가능성 환자 고지
- aspirin 못 쓰면 특정내역 필요:
  - 무효, 알러지, 저항성, 심한 부작용 등 명시
- 2종 병용은 반드시 삭감 → 기존 병원 의뢰 또는 f12

#### 항응고제
- rivaroxaban, apixaban, edoxaban, betrixaban 등
- 약마다 보험 기준 미묘하게 다르므로 전화 확인 후 처방
- 대부분은 f12 비급여 또는 기존 병원 방문 권고가 안전

#### 말초순환장애
- kallidinogenase 등은 I739 상세불명 말초혈관질환 코드

### 4-4. 이상지질혈증
- 코드: E78
- 초진:
  - 본원/타원 첫 진단이면 1년 내 검사 결과를 명세서에 기록
  - 기존 재처방이면 MX999에 ‘타원 이상지질혈증/고지혈증 약 재처방’
- 재진:
  - 단순 지속처방은 추가 기재 불필요
  - 증량/복합제 변경/타 성분 추가는 1년 내 검사결과 근거 필요
- 검사기록 예시:
  - 날짜
  - LDL 수치
  - 위험요인(흡연, 가족력, 고혈압 등)
- sim은 저녁, ato는 오전
- 위험군별 LDL 기준보다 30 이상 높으면 증량 또는 2단계 시작
- cd 대상 아님

### 4-5. 골다공증
- 키워드: alendronate, risedronate, ibandronate, 디카맥스, vitamin D, BMD, T-score, m818, 본비바
- 매 방문 시 명세서 작성 필요
- 최근 1년 이내 검사 결과 필요
- 최종 투약일이 검사일 이후 1년 초과하지 않게 주의
- 투여 간격:
  - alen 주 1회
  - rise 월 1회
  - iban IM(본비바) 3개월 1회
- 칼슘 단독 혹은 vitamin D+calcium 제제:
  - 골다공증 상병 있으면 인정
  - 없으면 100/100
- 상병 m818 기타 골다공증
- vitd는 시트라칼에프정
- 디카맥스는 같은 분류지만 칼슘 형태/양이 다른 일반약

### 4-6. 두통/편두통(만성)
- 코드: R51, G439
- 편두통은 cd 청구 가능
- eperisone BID 시 `m542 경추통` 필요
- 약속처방 3가지 중 하나만 사용:
  - 싸리움(flunarizine): 예방
  - 이미그란(sumatriptan): 급성
  - 크래밍(caffeine + ergotamine): 급성
- 크래밍 + 진통제 2종은 삭감
  - NSAIDs, ty, semi 중 1가지만 병용
- 크래밍은 분류상 해열/소염/진통제로 간주되므로, NSAIDs/ty/semi와 합산해 총 2종 넘지 않게 함
- 해열소염진통제 3종부터 삭감

### 4-7. 방광염 / 신우신염 / 소아 UTI / 귀두포피염
#### 방광염
- 코드: N300
- 1차: aug2
- 2차: cipro + 특정내역에 ‘1차 항생제에 호전 없어 처방함’
- cipro를 1차로 쓰면 N12-3 신우신염(의증) 추가
- IM genta 가능
- cipro 실패 시 모누롤산(fosfomycin) 1회 복용, 명세서에 근거 입력

#### 신우신염
- 코드: N12-2, N12-3
- cipro 2T BID 또는 levofloxacin 250mg QD 10일
- 초처방 후 2~3일 내 호전 없으면 전원
- IV anti: ns110 + 3cefaiv(ceftriaxone)
- d(디클로페낙) IM 보험 인정, 연결 m79108 삭제 가능
- 디클로페낙은 통풍/신장 통증 등에서도 추가 상병 없이 가능하다고 설명

#### 소아 UTI
- 보통 2일 정도 초처방
- cefa/augsy → 바난시럽 → 전원

#### 귀두포피염
- 예시: cefasy + lacto2 + typow 3일
- mupirocin 연고(bac) 추가 고려

### 4-8. 호흡기 질환
- 대표 코드:
  - J00 비인두염
  - J040 후두염
  - J060 후두인두염
  - J0180 부비동염
  - J0390 편도염
  - J303 비염
  - J069 상기도감염
  - J22 하기도감염
  - J209 기관지염
  - H6518/H669 중이염
- 오더 순서:
  1) 항생제
  2) NSAIDs/ty
  3) 대증약
  4) 위장약 등
  5) 외용제(pat1~3, cosy, tan)
- 대증약은 총 개수 제한과 동일 분류 중복 삭감 주의

#### 진해거담/콧물약 분류
- 성인
  - 기침: co, cosy
  - 가래: erdo, ac, drop
  - 콧물: suda
- 소아
  - 기침: dropsy, ambsy, umk
  - 가래: ac2

- 수면유도제 원하면 서카딘 가능
- 외국인 등록번호 불명/가명 의심 환자는 DUR 회피 가능성 있어 향정 처방 금지
- 신분증, 타원 향정 처방 이력 확인 필수

### 4-17. 근골격계 통증
- 키워드: 근통, 관절통, aceclofenac, diclofenac, celecoxib, eperisone
- pain 근골격계통증 묶음 참고
- d(디클로페낙 주사)만으로도 연결 m79108에 NSAIDs + ty/semi 보험 가능
- 더 디테일하게는 M코드 직접 입력해도 좋음
- epe(근이완제)는 관련 상병 필수
  - 현재 연결코드 m545 아래허리통증
  - 또는 m542 경추통
  - 자동입력 연결코드를 아예 지우면 삭감
- eperisone 50mg은 BID/TID 가능하나, 서방정 등 타 제제는 alt+D 용법 확인 후 보수적으로 BID
- celecoxib:
  - 효능효과만 보면 급성통증/월경통도 포함되지만 보험은 관절염/AS만
  - 확진 안 된 경우 m1388 기타 명시된 관절염
  - 또는 m170 양측 원발성 무릎관절증 사용
- set/semi 규칙:
  - NSAIDs + semi 가능
  - Ty + semi 삭감
  - NSAIDs + set 삭감
  - Ty + set 삭감

### 4-18. 빈혈
- 키워드: D50, Ferroba-U, 볼그레
- 검사:
  - Basic + anemia(철대사, smear)
  - anemia 검사 시 D50 자동입력, 삭제 금지
- IDA 과거력 없으면:
  - D50 배제된 상병으로 검사
  - 양성이면 이전/현재 기록 모두 해제
- 치료:
  - 4~6개월, 하루 1~2회
  - 1차 ferro
  - 위장장애 심하면 2차 볼그레액
  - 특정내역 혹은 명세서에 경구제 위장장애로 액상제 전환 사유 기재
  - 페리틴 등 수치가 보험 기준 충족해야 함

### 4-19. 주사 의뢰
- 외부 병원 정기 주사 의뢰 처리 순서:
  1) 의뢰서 확인
  2) 차트에 용법/용량/경로 기록
  3) 관련 상병
  4) sc(피하/근육내 주사 행위료) 입력
  5) 명세서 입력
- 바이알 사용은:
  - 1회용 앰플
  - 또는 재사용 책임을 기존 병원이 진다는 문구 있을 때만
- 예: 프로게스테론 주사
  - N979 상세불명 여성불임
  - 차팅: 프로게스테론 근주
  - 명세서: 프로게스테론 근주 시행
  - 처방: sc

### 4-20. 여성 갱년기 / 식욕부진 / 딸국질 / 눈떨림 / 글리아티린
- 리비알:
  - N951 + R232 같이 잡아야 삭감 안 됨
- 안젤릭정:
  - N951 메인 + R232 홍조
- 입맛 나는 약:
  - 메게이스 현탁액 1,1,5 정도 f12 비급여
  - 또는 trestan
  - 비급여 일반약이라 보험 청구 시 식욕부진 코드 + mosa 같은 위장약 추가
- 딸국질:
  - chlorpromazine, baclofen 가능
  - 위염 폴더에 있음
- 눈떨림:
  - Magnes 비급여 + 조제료 위해 소화제 같이
  - 상세불명 떨림 또는 마그네슘 결핍 코드
- 글리아티린:
  - F067 경도인지장애
  - 60세 이상에서만
  - 초처방은 환자 설명 필수

### 4-21. 생리통 / 부종 / 어지러움 / 일반약
- 생리통:
  - ‘월경통’ 검색
  - N946 상세불명 월경통
  - 진통제/진경제 가능
  - 처방 없이 확인서만 원하는 경우 secondary gain 주의
- 다이크로짇(thiazide):
  - 국소부종 상병에도 단독 사용 가능
- 어지러움:
  - R51 어지러움 vs H814 중추기원의 현기증은 둘 중 하나만
  - 젊은 환자에서 H814가 실비 가입에 불리할 수 있어 주의
  - R51: BPPV/단순 어지러움
  - H814: vasovagal syncope, 순환장애, ginkgo biloba 계열 등
- 사후피임약/발기부전/비만/여드름/탈모/고산병 등 일반약:
  - 보험약과 묶지 말고 일반 재접수 + 초진
  - F12 비보험도 안 됨
  - J00 감기 코드 사용(z/r 지양)
  - 진료실에서 뒤늦게 요구하면 일반 접수 하나 더 추가
- 일반약 예시: isotretinoin 등
- 환자 귀책 분실로 중복처방:
  - 일수 많이 겹치면 일반접수 + 비보험

### 4-22. 수족구/포진성 구협염
- 부모는 대개 ‘구내염’으로 표현
- 여름~가을 유행, 봄/겨울에도 가능
- 본원은 항바이러스제 지양, 대증치료 위주
- 감염 후 1주가 전염력 가장 높음, 격리 필요
- 기관 제출 서류:
  - 진료확인서에는 격리기간 문구 불가
  - 반드시 소견서/진단서로만
- 문구 예시:
  - 결석용: 최소 1주 격리 및 안정가료 필요
  - 출석용: 현재 정상체온/증상 소실, 전염력 적음

### 4-23. 독감
- NIP는 주로 10월 시작, 11~12월 주의보
- A/B형 유행, C형은 검사 거의 안 함
- 격리:
  - 해열 후 24시간까지
- oseltamivir:
  - 5일치 끝까지 복용
- peramivir IV:
  - 첫날 1회만
  - 나머지는 대증약
- 조플루자는 본원 미사용
- 타미플루캡슐:
  - 독감 폴더에서 직접 클릭
- 타미플루현탁액(tamisy):
  - alt+D로 kg당 용량 확인
  - 예: 16kg → 7.5ml BID → 총량 15, 2, 5
- 페라미플루(tamiiv):
  - 1회 2앰플로 치료 종료
  - tyiv + tamiiv 같이 사용 가능
- peramivir는 만 2세부터 가능하나, 본원은 중학생 이상을 기본 / 10세부터는 혈관 확보 가능할 정도면 시행

## 5. 수액 / 주사 / 보험·비보험 원칙 재정리
- 본원은 보험 적용되는 수액을 따로 사용하지 않음
- 주사제는 모두 IM 등록
- 주사제 여러 개 mix 시 1개만 보험, 나머지는 비보험코드
- 특수한 경우(알러지로 dexa+pheni) 제외하면 IM은 1종만
- 외용제/시럽/패치 일수는 Ctrl+T로 안 따라갈 수 있음
- 6세 이상 가루약 체크 필수
- 급여/비급여, 보험/일반접수, 100/100 여부를 항상 구분
- 신규 약제:
  - 뮤라틱주: 코드 aci, 8000원, ambroxol 대신 mix 가능
- 응급약:
  - epinephrine: 코드 epi, IM 폴더

## 6. 법령 / 행정 / 청구 관련 핵심
- 문서 상 제목으로 “필수로 알아야 할 법령 – 대리처방, 응급환자 이송 등등”이 있으나, 본문 텍스트 추출에서는 세부 항목이 길게 풀리진 않음. 다만 다음 행정 원칙은 문서 전반에 반복됨:
  - 일반약/비보험 약은 반드시 일반 재접수
  - 향정은 신분증/DUR 확인 필수
  - 수두는 보건소 신고
  - 서류 종류별로 청구/비청구 구분
  - 주사 의뢰는 의뢰서, 책임소재, 상병/명세서/행위료를 모두 맞춰야 함

## 7. AI가 반드시 기억해야 하는 병원 고유 룰
1. 오더 순서와 상병 순서를 통일한다.
2. IM/IV가 분리 등록되어 있지 않으므로 다중 주사제는 추가분 비급여 처리한다.
3. 호흡기 상병에서 loxoprofen은 삭감되므로 dexi/bru를 기본으로 생각하고, loxo는 요통 연결코드가 필요하다.
4. ‘배제된 상병’을 적극적으로 활용하되 메인상병으로 쓰지 않는다.
5. 만성질환은 재진/초진 구분을 의사랑 기본값 그대로 쓰면 틀릴 수 있다.
6. 일반약(사후피임약·ED·비만·탈모·여드름 등)은 보험진료와 섞지 않는다.
7. GI 약은 같은 분류 중복, H2/PPI 병용, motility drug 중복, 위염에 진통제 사용 등 삭감 포인트가 많다.
8. 호흡기 대증약은 분류 중복 삭감 규칙을 반드시 체크한다.
9. eperisone, celecoxib, propranolol, 글리아티린, 향정 등은 “효능효과만 보고” 처방하면 안 되고 보험/행정 조건을 같이 봐야 한다.
10. 외용제/시럽/패치/가루약/수액 mix는 의사랑 조작 실수 포인트가 많다.

## 8. 문서 내 포함되지만 텍스트로는 요약만 가능한 요소
- 다수의 스크린샷 예시(처방 예시, 상병/오더 예시, 진료실 이동 화면, 비청구 처리 화면 등)
- 기본 보험 주사제 / 비급여 추가 주사제 목록 이미지
- 서류 발급별 청구 여부 표/이미지
- 여러 카페 링크(실제 실무 참고자료)

## 9. 한 줄 SSOT
이 문서는 “24시열린의원에서 의사랑을 사용할 때, 질환별 처방 자체보다도 보험 인정 조건, 연결 상병, 일반/보험 접수 분리, 주사제/수액의 비급여 처리, 문서/서류/행정 루틴, 그리고 병원 내부의 오더 통일 규칙”을 우선 기억해야 하는 실무 중심 인수인계 문서다.
